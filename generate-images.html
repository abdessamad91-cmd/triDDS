Générateur d'images par catégorie

Génère automatiquement 21 pictogrammes pour les catégories de produits DDS et les uploade vers R2.

Comment ça marche : Chaque catégorie (Pâteux, Solvants, Aérosols...) reçoit un pictogramme coloré. Tous les produits d'une catégorie partagent la même image. L'upload se fait en mode Global (visible par tous les utilisateurs).
